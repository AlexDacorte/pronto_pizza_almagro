Pronto Pizzas & Empanadas Overview

Pronto Pizzas & Empanadas is a modern single-page application (SPA) designed for a fast, intuitive, and mobile-friendly food ordering experience in CABA, Argentina. The platform showcases a curated menu of pizzas and empanadas, enabling users to browse, customize, and place orders seamlessly.

Features

🍕 Dynamic Menu Browsing Explore a variety of pizzas and empanadas with detailed descriptions and pricing.

⚡ Fast Ordering Experience Optimized SPA performance for quick navigation and minimal load times.

📱 Responsive Design Fully optimized for mobile, tablet, and desktop devices.

Tech Stack

Frontend: React

Styling: TailwindCSS / Custom CSS

Routing: SPA-based routing (React Router)

Installation git clone https://github.com/AlexDacorte/pronto_pizza_almagro.git pronto_pizza_almagro npm install npm run dev

Location

CABA, Argentina Designed for local delivery and pickup experiences.
